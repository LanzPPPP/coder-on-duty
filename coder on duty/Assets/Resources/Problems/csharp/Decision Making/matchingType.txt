Selection statements: This is also known as selection structures which are used to execute particular statements whether the value of the given condition evaluates to true or false. 
Short-hand if else: Known as the ternary operator because it consists of three operands. It can be used to replace multiple lines of code with a single line. It is often used to replace simple if else statements.
If: This statement to specify a block of C# code to be executed if a condition is true.
Switch statement: Executes a block of statements based on the variable to be tested for equality against a list of values.
Else: Statement to specify a block of C# code to be executed if a condition is false.
